# Ax Service Hub — Concepto y Visión

## Qué es

Ax Service Hub es la capa tecnológica de Ax Holding Group.
No es una vertical de servicio — es la infraestructura horizontal que potencia todas las demás.

Funciona en dos dimensiones:

### 1. Uso interno
Herramientas, interfaces, sistemas y agentes diseñados para que Ax Holding Group opere
de forma autónoma, escalable y sin depender de estructura humana proporcional al volumen.

Cada vertical tiene sus propios flujos automatizados, agentes y sistemas construidos desde Ax Service Hub.

### 2. Oferta externa a clientes
Las mismas herramientas y sistemas que Ax usa internamente pueden ofrecerse como
servicio adicional a los clientes de las 5 verticales.

Ejemplos: dashboards de seguimiento para clientes de Prepcenter, herramientas de análisis de
listings para clientes de Agency, sistemas de gestión de proyectos para clientes de Design.

## El principio

Construimos para nosotros primero. Lo que funciona y tiene valor, lo ofrecemos al cliente.

## Estado actual

En construcción. Primera implementación: el Asistente Dev-IA (este espacio),
que actúa como co-piloto técnico para diseñar, construir y operar todo el ecosistema.

## Herramientas actuales del stack

- Claude / Anthropic: agentes, automatizaciones, asistentes
- Make.com: escenarios y automatizaciones
- HubSpot: CRM
- Apollo: prospección
- GitHub: versionado de código, skills y plugins
